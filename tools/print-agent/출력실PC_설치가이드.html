출력실 PC 인쇄 에이전트 자동시작 설치 가이드 PhotoCafe ERP · 출력대기 JDF+PDF 변환 · 작업지시서 자동인쇄 사용을 위한 1회 설정

왜 필요한가요? PhotoCafe 운영서버는 미국 클라우드(Railway)에서 동작합니다. 출력실 PC의 프린터/PDF저장 폴더에는 직접 접근할 수 없어, 출력실 PC에서 작은 프로그램(에이전트)을 띄워 다리 역할을 합니다. 이 에이전트가 꺼지면 JDF+PDF 자동변환·작업지시서 자동인쇄·로컬 폴더 PDF 저장이 모두 멈춥니다. 본 문서대로 한 번만 설정하면 PC를 켤 때마다 자동으로 시작·자동으로 재시작됩니다.

1준비물 확인

  • 출력실 PC (Windows 10/11)
  • 관리자 권한 계정 (UAC 「예」 클릭 가능해야 함)
  • Node.js LTS — 미설치 시 https://nodejs.org 에서 LTS 버전 설치
  • 본 가이드와 함께 받은 print-agent 폴더 (USB·공유폴더로 전달)

2폴더 복사

  1. 전달받은 print-agent 폴더 통째로 출력실 PC의 다음 위치에 붙여넣기:
    C:\PhotoCafe\print-agent\
  2. 붙여넣은 폴더 안에 다음 파일들이 보이는지 확인:
    • print-agent.js (본체)
    • watch-agent.vbs (자동재시작 워치독)
    • 작업스케줄러_등록.bat (1회 등록용)
    • run-print-agent.bat (수동 실행용 — 평소엔 사용 안 함)
경로 주의 경로에 한글이 들어가도 동작은 하지만, 권장 경로(C:\PhotoCafe\print-agent\)를 사용하면 문제 발생 시 진단이 빠릅니다.

3관리자 권한으로 등록 (핵심)

  1. 파일 탐색기에서 작업스케줄러_등록.bat 을 찾기
  2. 마우스 우클릭 → 「관리자 권한으로 실행」 선택
    그냥 더블클릭 ✗ 그냥 더블클릭하면 권한 부족으로 등록이 실패합니다. 반드시 우클릭 → 관리자 권한으로 실행 으로 진행하세요.
  3. UAC(사용자 계정 컨트롤) 창이 뜨면 「예」 클릭
  4. 검은 명령창에 다음 메시지가 나오면 성공:
    [성공] 작업 스케줄러에 등록되었습니다. - 다음 로그인부터 자동 실행됩니다.
    - 에이전트가 꺼지면 5초 후 자동 재시작합니다.
  5. 아무 키나 눌러 창을 닫음

4지금 바로 시작 (재부팅 없이)

등록 직후에는 아직 에이전트가 실행 전입니다. 즉시 실행하려면:

  1. Win + R 눌러 실행창 열기
  2. cmd 입력 후 Enter
  3. 검은 창에 다음 명령 붙여넣고 Enter:
    schtasks /run /tn "PhotoCafe 인쇄에이전트"
  4. SUCCESS: Attempted to run the scheduled task ... 메시지 확인

이후 PC를 켤 때마다 자동으로 시작되므로 이 단계는 다시 할 필요 없습니다.

5동작 확인

출력실 PC의 크롬에서 PhotoCafe 출력대기 페이지에 접속:

https://photocafe.co.kr/orders/print-queue

표시상태의미
에이전트 연결됨 (초록불 깜빡임) 정상 설정 완료. 자동변환·자동인쇄 모두 사용 가능
에이전트 꺼짐 (빨간불) 실패 3단계 다시 확인 → 관리자 권한 으로 .bat 실행했는지 점검
확인 중… 대기 5초 안에 위 둘 중 하나로 변경됨

「연결 재확인」 버튼을 누르면 즉시 다시 점검합니다.

6이상 동작 시 점검

증상점검 / 조치
등록 시 「Access is denied」 관리자 권한으로 실행하지 않은 것. .bat 우클릭 → 「관리자 권한으로 실행」
등록은 됐는데 「에이전트 꺼짐」 ① Win+R → taskschd.msc 작업 스케줄러 열기 → 「PhotoCafe 인쇄에이전트」 우클릭 → 「실행」
② 그래도 안 되면 Node.js 설치 확인 (node -v)
프린터 목록이 비어있음 출력실 PC의 「제어판 → 장치 및 프린터」에 프린터가 등록되어 있는지 확인. 등록 후 PhotoCafe 출력대기 화면 새로고침
자동 인쇄가 안 됨 PhotoCafe 「설정」 → 「자동 인쇄 사용」 체크 + 프린터명이 정확한지 확인
로그 보기 C:\PhotoCafe\print-agent\agent.log 파일을 메모장으로 열어 마지막 줄 확인

7해제 / 일시중지

  • 일시 중지: Win+R → taskschd.msc → 「PhotoCafe 인쇄에이전트」 우클릭 → 「사용 안 함」
  • 완전 삭제: 관리자 cmd 에서 schtasks /delete /tn "PhotoCafe 인쇄에이전트" /f
