================================================================
  PhotoCafe 로컬 인쇄 에이전트 - 설치 안내
================================================================

이 폴더의 파일들을 이용해 다른 PC에서도 PhotoCafe의
PDF 자동 저장 + 작업지시서 자동 인쇄 기능을 사용할 수 있습니다.


설치 순서 (5단계, 약 15분)
================================================================

[준비] 이 폴더 전체를 다른 PC에 복사
        예: USB 또는 네트워크로 복사
        권장 위치: C:\photocafe-agent\

[1단계] 1_프로그램_설치_안내.bat 더블클릭
        Node.js 와 SumatraPDF 다운로드 페이지가 자동으로 열립니다.
        두 프로그램을 다운받아 설치하세요 (다음·다음).

        Node.js     : https://nodejs.org/ko/download
                      → LTS 버전 다운로드 → 설치
        SumatraPDF  : https://www.sumatrapdfreader.org/download-free-pdf-viewer
                      → 64-bit installer 다운로드 → 설치

[2단계] 2_에이전트_시작.bat 더블클릭
        검은 콘솔 창이 뜨면서 에이전트가 시작됩니다.
        "[에이전트] 감지된 프린터 N개:" 메시지가 보이면 정상.
        ※ 이 창을 닫으면 인쇄/저장 기능이 멈춥니다.

[3단계] 브라우저로 PhotoCafe 접속
        https://photocafe.co.kr/orders/print-queue
        로그인 후 우측 상단 [설정] 버튼 클릭.

[4단계] 에이전트 설정 입력
        • 에이전트 저장 경로:   C:\PDF저장   (또는 원하는 경로)
        • 인디고 프린터:        그 PC의 프린터 선택
        • 잉크젯 프린터:        그 PC의 프린터 선택
        • 작업지시서 자동 인쇄: ON
        [설정 저장] 클릭.

[5단계] 3_자동시작_등록.bat 더블클릭 (선택, 권장)
        PC 부팅 시 에이전트가 자동 실행되도록 등록합니다.
        등록 후에는 검은 창이 안 뜨고 백그라운드에서만 동작.


파일 설명
================================================================

print-agent.js                  메인 에이전트 프로그램 (수정 금지)
1_프로그램_설치_안내.bat         Node.js + SumatraPDF 다운로드 안내
2_에이전트_시작.bat              에이전트 수동 실행
3_자동시작_등록.bat              Windows 시작 시 자동 실행 등록
자동시작_해제.bat                자동 시작 등록 제거
에이전트_백그라운드실행.vbs       자동 시작용 (수정 금지)
설치안내.txt                     이 파일


확인 방법
================================================================

에이전트가 실행 중인지 확인:
  웹브라우저에서 http://localhost:9199/printers 접속
  → 프린터 목록 JSON 이 보이면 정상 동작 중

PDF 변환 후 저장 위치 확인:
  설정한 경로 (예: C:\PDF저장\YYMMDD\)
  날짜 폴더가 자동 생성됨.

작업지시서 인쇄 확인:
  PDF 변환 완료 시 자동으로 1장 출력됨.
  안 나오면 프린터 큐 상태 확인 (오프라인 모드 해제).


문제 해결
================================================================

[Q] 검은 창에 "Node.js 가 설치되지 않았습니다" 메시지
[A] 1단계로 돌아가 Node.js 를 설치하세요.

[Q] 변환 후 PDF가 다운로드 폴더로 들어감
[A] 에이전트가 안 켜진 상태입니다. 2_에이전트_시작.bat 다시 실행.

[Q] 작업지시서가 안 나옴
[A] 1) SumatraPDF 설치 확인
    2) 프린터 큐 확인 → "오프라인으로 사용" 체크 해제
    3) 프린터 전원/용지/잉크 확인

[Q] 부팅했는데 자동 실행이 안 됨
[A] 3_자동시작_등록.bat 을 다시 실행.
    또는 Win+R → shell:startup → PhotoCafe_PrintAgent.vbs 존재 확인.
